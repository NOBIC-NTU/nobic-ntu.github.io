Introduction to FIJI - a course by NOBIC



The course is meant for microscopy users who are new to Image J/FIJI and to image processing in general, as well as for those who have already done their first steps in FIJI and want to gain a better orientation in the environment. We'll start from the very basics, so you don't need to be afraid if even you've never used FIJI before. By the time we are done you will be for example able to write simple macros to automate tedious image processing tasks.
