| Stereo Microscope Axio Zoom V.16



This is a parallel (CMO) fluorescence stereo microscope for observing larger specimens in fluorescence, transmitted or reflected light.

Available techniques:

  • Transmitted light imaging
  • Reflected light imaging (gooseneck illumination from above)
  • Fluorescence imaging

Objectives:

  • PlanNeoFluar Z 1x/0.25, FWD 56 mm
    [FWD = free working distance]
  • Zoom: 0.7x - 11.2x

Light sources:

  • HXP 200 C metal halide lamp (epifluorescence excitation)
  • CRI 90 LED (brightfield illumination)

Filter sets:

  • Zeiss set 49 - DAPI (G 365; 395; 420-470)
  • Zeiss set 38 - GFP (470/40; 495; 525/50)
  • Zeiss set 43 - DsRed (545/25; 570; 570-640)

Detectors and cameras:

  • Axiocam 503 colour camera (1936x1460 pixels, 4.54 µm/pixel)

Software:

  • ZEN Blue
